# open-device-manager

#### 介绍
设备管理前端 是用于与设备进行交互，请求设备拍照、录音，向设备推送语音、文本。没有后端代码。
它基于vue2 + Vant2 + @amap/amap-jsapi-loader + benz-amr-recorder等主流技术栈构建的简单设备管理前端。

#### 注：
高德地图配置要换成开发者的key和密钥;
html5录音功能需要使用 https 协议;

#### 项目启动

```bash
# 克隆代码
git clone https://github.com/chaojunHe/open-device-manager.git

# 安装依赖

npm install

# 启动运行

npm run serve
```
